Tentang saya

Atep Sutisna, S.Kom

S1 jurusan teknik infomatika kampus Universitas Bale Bandung yang lulus pada tahun 2022. Memiliki pengalaman dalam membuat aplikasi berbasis web menggunakan HTML, CSS, PHP dan JavaScript serta beberapa framework seperti CodeIgniter, Laravel, Bootstrap dan Tailwind. Aktif mengikuti seminar kampus dan luar kampus, aktif pula mengikuti pelatihan BPPTIK daring dan juga luring serta pelatihan-pelatihan daring lainnya seperti dicoding.

Kerja tim
